# ejerciciopoo-saiii

**Python POO**
- Una vez clonado el proyecto, en la terminal creamos un entorno 
```
    python -m venv env
```

- Y luego instalaremos **requeriments.txt**
```
    pip install -r requeriments.txt
```

- Ahora en sql creamos una base de datos que se llame "companydata"

- Luego de eso podemos ejecutar el proyecto
```
    python main.py
```
